# T-mux-Banner
customize your Termux terminal like a pro 

# Installation

$ apt update

$ apt upgrade

$ apt install git

$ git clone https://gitlab.com/Solozyco/t-mux-banner.git

$ cd T-mux-Banner

$ bash install

$ bash TXB.sh
